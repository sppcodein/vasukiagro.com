# VASUKI AGRO CHEMICALS Block Theme

A custom WordPress block theme for VASUKI AGRO CHEMICALS LLP, a manufacturer of plant growth promoters using bio materials.

## Features

- **Full Site Editing (FSE) Support**: Create and edit your entire site using the WordPress block editor
- **Custom Block Patterns**: Pre-designed block patterns tailored for agricultural businesses
- **Custom Block Styles**: Enhanced styling options for WordPress core blocks
- **Responsive Design**: Optimized for all devices - mobile, tablet, and desktop
- **Performance Optimized**: Fast loading with optimized assets and lazy loading
- **Engaging Animations**: Subtle animations to enhance user experience
- **Agriculture-Focused Design**: Color palette and styles designed specifically for agricultural businesses

## Templates Included

- Home Page
- About Us
- Products
- Team
- Contact
- Single Post/Page
- Archive Pages

## Custom Block Patterns

The theme includes several custom block patterns designed specifically for agricultural businesses:

1. **Product Features Grid**: Showcase product features with icons and descriptions
2. **Farmer Testimonials**: Display testimonials from farmers using your products
3. **Call to Action Banner**: Encourage visitors to contact you or explore products

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher

## Installation

1. Upload the theme folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Go to 'Appearance > Editor' to start customizing your site

## Customization

### Colors and Typography

The theme uses a nature-inspired color palette designed for agricultural businesses. You can customize these colors in the Site Editor:

1. Go to 'Appearance > Editor'
2. Click on 'Styles' in the top-right corner
3. Adjust colors, typography, and other global settings

### Templates and Patterns

1. In the Site Editor, you can modify any of the included templates
2. To use block patterns, click the '+' icon in the editor and look for the 'Vasuki Agro' category
